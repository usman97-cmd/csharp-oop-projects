# Employee Management System

A C# Console Application demonstrating Object-Oriented Programming concepts.

## Features

- Add Employee
- Display Employees
- Search Employee
- Remove Employee

## OOP Concepts Used

- Encapsulation
- Constructors
- Inheritance
- Runtime Polymorphism
- Virtual & Override
- List Collection

## Technologies

- C#
- .NET
- Console Application
