# Employee Management System (C# Console App)

A simple console-based Employee Management System built using C#, OOP, Collections, and LINQ.

🚀 Features

Add Developer

Add Manager

Show All Employees

Filter IT Employees

Show High Salary Employees

Search Employee By ID

Show Employee Names

Sort Employees By Salary

Display Statistics

Prevent Duplicate IDs

🧠 Concepts Used

Encapsulation

Inheritance

Abstraction

Polymorphism

Collections (List<T>)

LINQ

Exception Handling (basic)
